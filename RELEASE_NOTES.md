# New API Manager - 发布说明

## 🎉 v0.0.1 首次发布 (2024-07-05)

### 📱 应用简介

**New API Manager** 是一款专为 new-api 项目设计的跨平台移动监控管理应用。支持统一监控和管理多个 new-api 实例，提供实时状态监控、系统统计、异常日志查看等功能。

### ✨ 核心功能

#### 🔧 实例管理
- ✅ 添加/删除多个 new-api 实例
- ✅ 实例连接状态检测
- ✅ 支持自定义实例名称和描述
- ✅ 安全的 Access Token 管理

#### 📊 系统监控
- ✅ 实时健康状态检查
- ✅ 系统版本信息显示
- ✅ KEY 数量统计
- ✅ 用户数量统计
- ✅ 支持模型数量统计
- ✅ 最近访问的 KEY 列表
- ✅ 登录用户活动监控
- ✅ 调用模型使用情况

#### 📈 数据分析
- ✅ 模型调用 Token 使用统计
- ✅ 模型调用次数统计
- ✅ 访问 IP 地址列表
- ✅ 支持域名反向查询
- ✅ 可视化数据图表展示

#### 🛠️ 应用特性
- ✅ 现代化 Material Design 界面
- ✅ 响应式布局适配
- ✅ 智能错误处理
- ✅ 离线数据缓存
- ✅ 中文本地化支持

### 🚀 平台支持

#### Android
- ✅ 支持 Android 6.0+ (API 23+)
- ✅ 适配各种屏幕尺寸
- ✅ 支持深色/浅色主题
- ✅ 后台数据同步

#### iOS
- ✅ 支持 iOS 12.0+
- ✅ 适配 iPhone 和 iPad
- ✅ 支持系统主题切换
- ✅ 原生导航体验

#### HarmonyOS
- ✅ 支持 HarmonyOS 3.0+
- ✅ 鸿蒙原生组件
- ✅ 分布式数据同步
- ✅ 智能场景联动

### 🔧 技术架构

#### 前端技术栈
- **框架**: React Native 0.80.1
- **语言**: TypeScript 4.9.5
- **导航**: React Navigation 6.x
- **状态管理**: React Hooks + Context
- **存储**: AsyncStorage
- **网络**: Axios + 自定义拦截器

#### 开发工具
- **构建工具**: Metro Bundler
- **代码检查**: ESLint + Prettier
- **类型检查**: TypeScript
- **测试框架**: Jest + React Native Testing Library

#### 数据处理
- **API 集成**: RESTful API 调用
- **错误处理**: 全局错误边界
- **数据缓存**: 智能缓存策略
- **离线支持**: 模拟数据降级

### 📋 安装要求

#### Android
- 最低系统版本: Android 6.0 (API 23)
- 存储空间: 50MB
- 网络权限: 必需

#### iOS
- 最低系统版本: iOS 12.0
- 存储空间: 50MB
- 网络权限: 必需

#### HarmonyOS
- 最低系统版本: HarmonyOS 3.0
- 存储空间: 50MB
- 网络权限: 必需

### 🚀 安装指南

#### Android APK
1. 下载 `NewAPIManager-v0.0.1.apk`
2. 允许"未知来源"应用安装
3. 点击安装并等待完成
4. 启动应用开始使用

#### iOS IPA
1. 使用 TestFlight 或企业分发
2. 信任开发者证书
3. 安装应用
4. 启动应用开始使用

#### HarmonyOS HAP
1. 通过华为应用市场安装
2. 或使用开发者模式安装 HAP 包
3. 授予必要权限
4. 启动应用开始使用
